# study-test-ui 组件库

### 快速开始

#### 安装组件库
```bash
npm i study-test-ui
```

#### 引用组件库

```javascript
//全部引入
import 'study-test-ui/dist/css/index.css';
import STYUI from 'study-test-ui';
Vue.use(STYUI)

//按需引入

import 'study-test-ui/dist/css/demo.css';
import {Demo} from 'study-test-ui'
Vue.use(Demo)

```
